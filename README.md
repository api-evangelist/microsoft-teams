# Microsoft Teams (microsoft-teams)
Microsoft Teams is a collaboration platform that combines workplace chat, meetings, file storage, and application integration. It provides APIs for building custom integrations and extending Teams functionality.

**URL:** [Visit APIs.json URL](https://www.microsoft.com/en-us/microsoft-teams/group-chat-software)
